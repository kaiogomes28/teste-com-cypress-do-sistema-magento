# Teste QA - Automacao E2E Web (Magento)

Automacao de testes funcionais end-to-end com Cypress no site de demonstracao:

- https://magento2-demo.magebit.com/

## Escopo automatizado

Os fluxos abaixo foram implementados:

1. Cadastro de usuario
2. Login
3. Adicionar produto ao carrinho
4. Finalizacao de compra (checkout)

Os testes estao no arquivo:

- `cypress/e2e/magento-qa-fluxos.cy.js`
